About Us

Hi! We are high schoolers at Pinewood in Los Altos and we enjoy programming. We would love to teach this to kids (ages 8-15)! We think that programming should be fun and strive to teach it that way. This is not aimed to be a professional summer camp but rather a fun, small summer camp tutoring service.

NOTE: We are currently expecting all camps to be online this summer, so we will be using Zoom. However, if in-person camps will be possible, we will try to find a school/location to do in-person camps.

Python Camp

Mon-Fri: 9AM - 12:30PM PT

Target audience: kids 8-15 years old

During this camp you will learn what Python is and how to program with it! No programming experience is required.

You will learn about programming concepts like variables, loops, if statements, functions, lists, etc. We will dive into turtle and have a competition on who can make the best art! Throughout the week, we will do projects to keep everybody engaged. We end with a final project that you can show everyody!

Requirements:

  • A computer

Pricing: $195 a week

Web Development Camp

Mon-Fri: 9AM - 12:30PM PT

Target audience: kids 8-15 years old

During this camp you will learn how to make your own websites using HTML, CSS, and Javascript. No programming experience is required!

You will learn what HTML, CSS, and Javascript are and how to use them! You will also learn programming concepts like variables, loops, if statements, functions, lists, etc. We will even dive into the framework Bootstrap! Multiple websites will be made throughout the week and we end with a big final website!

Requirements:

  • A computer w/ Atom installed (instructions follow later)

Pricing: $195 a week

iOS Development Camp

Mon-Fri: 9AM - 12:30PM PT

Target audience: kids 9-15 years old

During this camp you will learn how to make your very own iOS apps using Apple's Swift and Xcode. No programming experience is required.

You will learn what Swift and Xcode are. You will also learn about programming fundamentals like variables, loops, if statements, functions, lists, etc. Throughout the week, multiple apps will be built to show off!

Requirements:

  • A macbook w/ Xcode installed (instructions follow later)

Pricing: $195 a week

Procreate Camp

Mon-Fri: 9AM - 12:30PM PT

Target audience: kids 8-13 years old

Do you want to know how to use the app Procreate to draw? During this camp you will learn the ins and outs of Procreate. You do not have to be a star at art to make fun projects with this app.

You will learn how to navigate Procreate. You will also learn about different brushes, layers, symmetry, liquidfication (marbling), etc. It will be combined with fun projects!

Requirements:

  • An ipad
  • the app Procreate (around ten dollars)
  • Optional: Apple Pencil

Pricing: $195 a week
